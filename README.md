## mysql config✔️
Add the DB details in .env file

```
DB_CONNECTION=mysql
DB_HOST=localhost
DB_PORT=3306
DB_DATABASE=
DB_USERNAME=
DB_PASSWORD=

```
## firebase storage config✔️

Add fireabse.js initializing firebase in /public/js

```
var firebaseConfig = {
    apiKey: "",
    authDomain: "",
    databaseURL: "",
    projectId: "",
    storageBucket: "",
    messagingSenderId: "",
    appId: "",
    measurementId: ""
};

const Firebase = firebase.initializeApp(firebaseConfig);
firebase.analytics();

```
## To create migrations✔️
```
php artisan migrate:fresh
```
## To serve the application🎃

```

php artisan serve
```
